# 0x12: JavaScript Fundamentals

This repository contains solutions to foundational JavaScript tasks using Node.js.

## Requirements

- All files must be executed using Node.js (version 24.x).
- All variables must be declared using `const` or `let`. The `var` keyword is forbidden.
- All output must be printed using `console.log(...)`.

## Tasks Included

- **0-javascript_is_amazing.js**: Demonstrates using `const` and printing a string.
- **1-multi_languages.js**: Prints multiple strings using multiple `const` variables.
